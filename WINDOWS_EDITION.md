## 🆘 TROUBLESHOOTING

### ❓ Script không chạy?
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### ❓ Lỗi "Access Denied"?
→ Phải chạy PowerShell với quyền **Administrator**

### ❓ Ping vẫn cao?
→ Kiểm tra:
1. ISP/Router
2. Dùng Ethernet (không phải Wi-Fi)
3. Tắt VPN/Proxy
4. Khởi động lại máy

### ❓ Muốn hoàn tác?
→ Tìm folder backup: `C:\Users\[Tên]\NetworkOptimizer-Backup-...`
→ Restore các file `.reg`
→ Khởi động lại
